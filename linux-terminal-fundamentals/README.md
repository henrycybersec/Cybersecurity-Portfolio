# Linux Terminal Fundamentals Lab

## Lab Type
Operating System / Command Line Practice

## Objective
To gain hands-on experience with Linux terminal navigation and basic file system management.

## Environment
- Kali Linux virtual lab

## Commands Practiced
- pwd
- ls / ls -l
- cd
- mkdir
- touch
- cat
- cp
- sudo

## Tasks Performed
1. Navigated directories using cd.
2. Created folders and files.
3. Viewed file contents using cat.
4. Copied files between directories.
5. Observed permission restrictions requiring sudo.

## Skills Demonstrated
- Linux command-line navigation
- File system operations
- Basic privilege awareness
- Terminal workflow confidence

## Outcome
Completed foundational Linux terminal operations forming the base for future cybersecurity tasks such as enumeration and scanning.
