# Log Analysis Lab

## Lab Type
Host-based Monitoring / Security Log Investigation

## Objective
To analyse system and authentication logs on both Windows and Linux systems in order to understand how security events are recorded and monitored.

## Environment
- Windows system (Event Viewer)
- Kali Linux system logs

## Activities Performed

### Windows Security Log Investigation
- Opened Event Viewer
- Navigated to Windows Security logs
- Observed successful login events
- Observed logoff events
- Identified importance of Event IDs

### Linux Log Exploration
Command used:
ls /var/log

- Identified system log locations
- Observed different log categories:
  - boot logs
  - authentication logs
  - service logs

## Skills Demonstrated
- Log monitoring fundamentals
- Host-based investigation mindset
- Understanding authentication events
- SOC analysis foundation

## Outcome
Developed basic log analysis skills required for security monitoring and incident detection.
