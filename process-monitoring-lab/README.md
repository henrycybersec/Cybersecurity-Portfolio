# Process Monitoring Lab

## Lab Type
Endpoint Monitoring / System Activity Observation

## Objective
To observe running processes in a Linux system and understand how analysts monitor system activity and privilege levels.

## Environment
- Kali Linux terminal

## Command Used
ps aux

## Activities Performed
- Listed all running processes
- Observed root vs user processes
- Identified background system services
- Understood process visibility in Linux

## Skills Demonstrated
- Endpoint monitoring basics
- Process inspection awareness
- System activity observation mindset
- Privilege context understanding

## Outcome
Developed foundational skills for detecting suspicious system activity and understanding host behaviour.
